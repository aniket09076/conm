Unit 3: Numerical Methods

Explore methods to find solutions when exact answers are difficult or impossible to calculate.

Key Concepts:

Interpolation
  • Lagrange Interpolation: Estimates unknown values between known data points.
  • Inverse Interpolation: Finds the value of an independent variable when the dependent variable is given.
Finite Differences
  • Forward Differences: Calculates the difference between successive data points.
  • Backward Differences: Works in the opposite direction to forward differences.
  • Divided Differences: Handles unevenly spaced data points.
Difference Tables
  • Forward and Backward Difference Tables: Organizes differences for interpolation.
  • Divided Difference Table: Organizes divided differences for unevenly spaced data points.
Newton's Methods of Interpolation
  • Newton's Forward Difference Method: Estimates values for evenly spaced data points.
  • Newton's Backward Difference Method: Focuses on values at the end of the dataset.
  • Newton's Divided Difference Method: Used for unevenly spaced data points.
Numerical Integration
